概要及目录
----------

    节能系统单元设备名称及功能概述:
    
    人体传感器：
        也称人体感应传感器，用于检测环境中是否有人出现，准确来说是检测环境内是否有物体 **移动** 

        * 工作状态: 绿色灯间隔5秒亮一次
        * 检测到移动:  检测到人体后，模块红色灯亮
        * 未检测到移动: 红色灯不亮

    温度传感器：
        检测所处位置的环境温度。
        
        * 温度采集周期: 3秒
        * 采集精度: 0.1
